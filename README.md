## This Is Why There Is ABS
A data-driven analysis of 1.4 million ball-strike
calls from the 2022-2025 MLB seasons.

## Contents
python/ABS.ipynb — data pipeline and analysis
rmd/ABS_WhitePaper.Rmd — white paper source
rmd/ABS_WhitePaper.html — rendered white paper

## Data
Raw data pulled from MLB Statcast via pybaseball.
called_clean_classified.csv generated by the Python pipeline.

## Setup
The CSV data files are not included in this repository 
due to file size. Run ABS.ipynb in the python folder 
to generate them locally.
